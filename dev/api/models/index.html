    数据模型¶

    所有 API 响应均使用 Pydantic v2 模型。

    认证¶

    TokenResponse ¶

    Bases: BaseModel

    登录成功后返回的令牌对。

    access_token instance-attribute ¶

    access_token: str

    访问令牌。

    refresh_token instance-attribute ¶

    refresh_token: str

    刷新令牌。

    token_type instance-attribute ¶

    token_type: str

    令牌类型，通常为 bearer。

    expires_in instance-attribute ¶

    expires_in: int

    访问令牌有效期（秒）。

    UserProfile ¶

    Bases: BaseModel

    用户信息。

    user_id instance-attribute ¶

    user_id: str

    用户 UUID。

    username instance-attribute ¶

    username: str

    用户名。

    display_name instance-attribute ¶

    display_name: str

    显示名称。

    role instance-attribute ¶

    role: str

    角色，如 admin、user、bot。

    is_active instance-attribute ¶

    is_active: bool

    账号是否启用。

    leader_expires_at class-attribute instance-attribute ¶

    leader_expires_at: datetime | None = None

    领队权限过期时间，None 表示无领队权限。

    created_at instance-attribute ¶

    created_at: datetime

    账号创建时间。

    updated_at instance-attribute ¶

    updated_at: datetime

    账号最后更新时间。

    分页¶

    PaginatedResponse ¶

    Bases: BaseModel, Generic[T]

    分页响应。

    items instance-attribute ¶

    items: list[T]

    当前页数据列表。

    total instance-attribute ¶

    total: int

    符合条件的总数。

    limit instance-attribute ¶

    limit: int

    每页数量。

    offset instance-attribute ¶

    offset: int

    偏移量。

    题目¶

    QuestionSummary ¶

    Bases: BaseModel

    题目摘要（列表项）。

    question_id instance-attribute ¶

    question_id: str

    题目 UUID。

    source instance-attribute ¶

    source: QuestionSource

    源文件信息。

    category instance-attribute ¶

    category: str

    分类：none、T（理论）、E（实验）。

    status instance-attribute ¶

    status: str

    状态：none、reviewed、used。

    description instance-attribute ¶

    description: str

    题目描述。

    score class-attribute instance-attribute ¶

    score: int | None = None

    分值。

    author class-attribute instance-attribute ¶

    author: str = ''

    命题人。

    reviewers class-attribute instance-attribute ¶

    reviewers: list[str] = Field(default_factory=list)

    审题人列表。

    tags class-attribute instance-attribute ¶

    tags: list[str] = Field(default_factory=list)

    标签列表。

    difficulty class-attribute instance-attribute ¶

    difficulty: dict[str, DifficultyValue] = Field(
    default_factory=dict
)

    难度评估，key 为评估标签（如 human）。

    allow_auto_reviewer class-attribute instance-attribute ¶

    allow_auto_reviewer: bool = False

    是否允许自动添加审阅人。

    created_by class-attribute instance-attribute ¶

    created_by: str | None = None

    创建者用户 UUID。

    created_at instance-attribute ¶

    created_at: datetime

    创建时间。

    updated_at instance-attribute ¶

    updated_at: datetime

    最后更新时间。

    QuestionDetail ¶

    Bases: QuestionSummary

    题目详情，包含资源和关联试卷。

    tex_object_id instance-attribute ¶

    tex_object_id: str

    TeX 文件对象存储 ID。

    assets class-attribute instance-attribute ¶

    assets: list[QuestionAssetRef] = Field(default_factory=list)

    附属资源列表。

    papers class-attribute instance-attribute ¶

    papers: list[QuestionPaperRef] = Field(default_factory=list)

    所属试卷列表。

    QuestionSource ¶

    Bases: BaseModel

    题目源文件信息。

    tex instance-attribute ¶

    tex: str

    TeX 主文件路径。

    DifficultyValue ¶

    Bases: BaseModel

    难度评估值。

    score class-attribute instance-attribute ¶

    score: int = Field(ge=1, le=10)

    难度分值 (1-10)。

    notes class-attribute instance-attribute ¶

    notes: str | None = None

    备注。

    updated_by class-attribute instance-attribute ¶

    updated_by: DifficultyUpdatedBy | None = None

    最后更新者。

    DifficultyUpdatedBy ¶

    Bases: BaseModel

    难度评估的更新者信息。

    user_id instance-attribute ¶

    user_id: str

    用户 UUID。

    username instance-attribute ¶

    username: str

    用户名。

    display_name instance-attribute ¶

    display_name: str

    显示名称。

    QuestionAssetRef ¶

    Bases: BaseModel
